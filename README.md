# codeceptJS - puppeteer boilerplate
## To run the tests:
1. Build package.json
2. node run.js

## Reference Links
[CodeceptJS](https://codecept.io/ "https://codecept.io/")

## Prerequisites
1. Install Node.JS version > 12
2. Javascript

---

## Addressed most common automation issues
- Test flakiness
- Parallelization
- Cross browser with native apps
- Image comparison
- Mocking
